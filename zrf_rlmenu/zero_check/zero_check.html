Description: Checks if the bin is empty
FUNCTION ZERO_CHECK.
*"----------------------------------------------------------------------
*"*"Local interface:
*" IMPORTING
*" VALUE(I_LTAP) LIKE LTAP STRUCTURE LTAP
*" EXPORTING
*" VALUE(O_LTAP) LIKE LTAP STRUCTURE LTAP
*" VALUE(O_SCREEN_FCODE) LIKE SY-UCOMM
*"----------------------------------------------------------------------

* Global data declarations

  CLEAR   LTAP_ZERO.

  MOVE I_LTAP TO LTAP_ZERO.

  CALL FUNCTION 'CALL_SCREEN_0410'
       IMPORTING
            O_SCREEN_FCODE = SCREEN_FCODE
            O_LTAP         = O_LTAP.

  CASE SCREEN_FCODE.

    WHEN FCODE_CANC.
      MOVE I_LTAP TO O_LTAP.
      O_SCREEN_FCODE = SCREEN_FCODE.

    WHEN OTHERS.
      MOVE LTAP_ZERO TO O_LTAP.

  ENDCASE.

ENDFUNCTION.
